Mashr

Mashr is an easy-to-use data pipeline framework that orchestrates moving data from external sources into a single destination database. Itâ€™s meant for small software applications who have their data spread out in a variety of external sources which they want to be able to aggregate and perform analysis on.

Mashr works by launching a data pipeline in the cloud with archiving, failover, and monitoring infrastructure out of the box. With the command `mashr deploy` and configuration file, a user can get started integrating different sources of data (from REST api's, psql databases, and more) into a single destination database for analysis. It uses Google Cloud Storage for backup and failover, Google Cloud Functions for loading data into BigQuery, and GCE instances hosting Embulk for plugin based bulk data loading. Mashr provides an easy to use CLI, is built with Node.js and uses Docker to host Embulk on the GCE instances.
