// Recent Project

Mashr

Mashr is an open source data pipeline orchestration and monitoring framework on GCP for small applications.

Mashr provides the convenience of deploying an entire data pipeline with best practices. It is a CLI application that was built using Node.js and published to npm for easy install. It uses a variety of GCP services, Docker, and a tool called Embulk for data extraction.
